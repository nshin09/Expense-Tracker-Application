Setup to get the app running on your device.

1. Locate the desired directory for the app (can be anywhere), right click and select the option “Open Git Bash Here”
A Git Bash Terminal Window will appear, enter the follow commands:
  - git clone https://github.com/nshin09/Expense-Tracker-Application.git
  - cd Expense-Tracker-Application
  - pip install -r requirements.txt 
  - flask run
 
2. On the web browser of your choice, search the following:
  - localhost:5000

3. The app should now be successfully running!
  - On subsequent runs, it is only necessary to open git bash in the directory of the app and run the command “flask run”.
